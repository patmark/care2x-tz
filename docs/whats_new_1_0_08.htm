The following are new in the Care2x beta 1.0.08 release:

A. Paginator for search results

  • Search results are now paginated with certain number of rows per page.
  • The number of rows shown per page is admin configurable (needs direct db access).
  • The user can turn pages forward or backward by clicking on the "Previous" or "Next" links.
    • B. Advanced sorting of search results

    • The search results can be sorted by the user by just clicking on a column's title.
    • The second click on the same column's title reverses the sort order.
    • Very easy to use.

    C. "Smarter" search for persons

    • The search can now detect user's input if it is a lastname, first name, date of birth, or combination of them
    • Search algorithm was optimized for a high volume data (latest test in a 1.5+ million persons databank)

    D. Much more stable than previous versions

  • Lots of debugs done and patches made.
  • The Nagoya Dicom Viewer was debugged.
  • Most active bug reporter was Thomas Wiedmann (DE).
  • Most active debugger and patch maker was Agus Ramdan (IN)
    • E. Operation logbook enhancement
  • The menu buttons of the OR logbook module are now "browser-independent".

    • F. Language versions

    • Spanish and Brazilian-Portuguese versions are almost complete.
    G. True type font support
  • True type capability is detected automatically and TTF fonts are used in patient's labels and wristbands.
    • H. Help pages are enhanced and expanded.
