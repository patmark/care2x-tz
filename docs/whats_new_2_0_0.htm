The following are new in the 2nd Generation beta 2.0.0 version:
NOTE:

    This release is not officially recommended to update the existing Deployment 1.1 !

    Inofficially, you can do an update under certain conditions described in this document.
    If you are unsure and want to be on the safe side, please wait for the release version that is intended to update the Deployment 1.1.

A. PostgreSQL support

  • PostgreSQL database is now supported in addition to the mySQL.
  • Test runs were done with PostgreSQL 7.2
  • The auto-installer can install either in mySQL or PostgreSQL

    • B. Patches

  • The recent Patch D-1.1.0.1 was also incorporated in this release.
  • New patches specific to display of TTF fonts in Arabic language (Walid Fathalla)

    • C. New Features and Improvements

  • Smarty 2.6.0 Template System (added by T. Wiedmann)
  • Improved configurator for departments (Kurt Brauchli)
  • Improved immunization module and automatic zip code input (Jean-Philippe LIOT)
  • Added button on the admission data page to go directly to registration data page

    • D. CDS - Central Data Server and HXP Server

    This version has an experimental support for Central Data Server using the HXP exchange protocol. For details please go here.

    E. Language and Help files

  • Chinese-simplified
  • German
  • Spanish

    • F. Coding System

  • Spanish ICD10 and OPS301 (Dr. Montemayor)
