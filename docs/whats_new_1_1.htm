The following are new in the Care2x deployment (mysql) 1.1 release:

A. "Deployment" status

  • Care2x uses the term "deployment" instead of "stable".
  • Why "deployment" ?
    Care2x is actually quite stable that it does deserve to be labeled as stable, but some language versions are not 100% complete. However, these incomplete language versions DO NOT hinder the reliability of the application.
  • Why mySQL?
    This version is tested to work for mySQL only.
  • Users can now choose to deploy Care2x as-is ("out-of-the-box") or adapt it to their local situation and needs.
  • Untranslated items are just displayed in the default english language.
  • Adding the translation is as simple as copying the language files or images to their proper subdirectories. This can be done even during normal use of the application without adverse effect.
