The following are new in the 2nd Generation beta 2.0.1 version:
NOTE:

    This release is not officially recommended to update the existing Deployment 1.1 !

    Inofficially, you can do an update under certain conditions described in this document.
    If you are unsure and want to be on the safe side, please wait for the release version intended to update the Deployment 1.1.

A. New Features and Improvements

  • Smarty templates for the main structure and lots of important GUI elements. The template is user-selectable at runtime via the GUI.
  • CSS based GUI design themes. 7 CSS designs are included. A theme is user-selectable at runtime via the GUI. Adding new themes is very easy, here is the how-to document.
  • Other hospitals' patient ID entry.
  • PLUGIN Manager. Now, a simple plug-in/plug-in system is implemented in Care2x. Several plugins are also available online for download.
  • Table Manager Plugin for managing the address and immunization types tables (by Dr. Jean-Philippe LIOT)
  • Automatic immunization schedule generator Plugin (by Dr. S.B. Bhattacharyya)
  • Adodb version 4.21

    • B. Arabic PDF generation support

  • New scripts and fonts specific to generation of PDF documents in Arabic language (Walid Fathalla)

    • C. PostgreSQL support

  • Additional debugs for PostgreSQL support
  • More test runs were done with PostgreSQL 7.2

    • D. Language and Help files

  • Bosnian (by Emir Prcic)
  • Serbian (by Petar Savic)
  • Greek (by Dimitris Dimitriadis)
  • Farsi (by Sajad)

    • E. Coding System

  • Bosnian/Latin ICD10 (by Emir Prcic)

    • F. HL7 Toolkit classes

  • Available from the CVS. Developed by Docu Dokter.
    • G. CDS - Central Data Server and HXP Server
    This version has an experimental support for Central Data Server using the HXP exchange protocol. For details please go here.
