How to translate Care2x

The iso code for simplified chinese = "zh" will be used for the examples below. You can also use the chinese traditional "zh-tw" if necessary.

Other language codes are listed at the bottom of this page.

Where to start?

You can start by translating the following files:

/language/en/lang_en_indexframe.php
/language/en/lang_en_startframe.php
/language/en/en_copyrite.php
/language/en/lang_en_stdpass.php
/language/en/lang_en_invalid-access-warning.php
/language/en/en_credits.php
Step 1. Create a new directory "zh" inside the /language/ directory.

Step 2. Copy the above listed files from /language/en/ to /language/zh/

Step 3. Rename the files by replacing the "en_" part with "zh_" .

Step 4. Copy the following files

/language/en/tabs.php
/language/en/roots.php
Step 5. Open the "tabs.php" file and change the following:

Set formal name of your language. Use the original form, not the "english" form. E.g "Cesky" for Czech language,not "Czech".

$lang_name='English'; => $lang_name='Chinese-simplified';
Set the code used for tagging the language files:

$lang_iso_code='en'; => $lang_iso_code='zh';
Set the character set:
$lang_charset='iso-8859-1' => $lang_charset='gb2312';
Step 6. Open the "roots.php" file and change the ff:
$top_dir='language/en/' => $top_dir='language/zh/';
If you did all correctly, you will notice that after refreshing your browser, the Chinese language item is now in the language selector on the left main menu.

What to do next?

You can send a file to admin(at)care2x.com everytime you finished it or you can upload it to the project's CVS and notify admin(at)care2x.com.

OK, then what next?

You decide which files you translate next.

OK, I've finished all files. What to do next?

Send a colored photo of yours with a short info about yourself (in your language and in english) to admin(at)care2x.com. We will add you to the credits page.

OK, everything sent. What to do next?

Nothing more. Use and enjoy your new Care2x.
