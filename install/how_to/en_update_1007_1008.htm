Updating (beta 1.0.07) to (beta 1.0.08)

NOTE: The following update procedure is valid ONLY if you have a working 1.0.07 installation already.

1. Uncompress the CARE 2002 beta 1.0.8 scripts into a temporary directory.

2. Copy all the files EXCLUDING the /include/inc_init_main.php file into your working installation.

Tip: If you have a file manager that compares file dates, use it to copy only the modified files.
3. Start your phpmyadmin (or any other mysql admin program).

4. Run the following sql script:

install/sql/db_update_1007_1008.sql
5. That was all!

Note: I have used the sql script to update few online demos . It worked well. However, it was not an exhaustive test, so please notify me at once if any problem occurs. Thanks. Elpidio
